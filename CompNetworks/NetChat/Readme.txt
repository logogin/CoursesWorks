Readme for Java Chat.
---------------------

client.html
-----------
In order to install this Java Chat all you have to do 
is to change a value of two parameters:
1.param name = "host" value = "localhost"
2.param name = "port" value = "8000"
Simply change value of host to a desireable host name. 
Also you can change value of port.

server.html
-----------
Simply change value of port to the same value of client.
